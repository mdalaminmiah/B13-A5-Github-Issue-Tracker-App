1️⃣ var vs let vs const
var: The old way. It is function-scoped and can be hoisted, which often leads to unpredictable bugs.

let: The modern way for variables that will change. It is block-scoped, meaning it only exists within the curly braces {} where it was defined.

const: The modern way for variables that won't change. Like let, it is block-scoped, but it prevents you from accidentally reassigning a new value to the variable.
